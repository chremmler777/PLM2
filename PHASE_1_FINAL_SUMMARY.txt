🎓 ARCHITECTURE HIGHLIGHTS

Backend → Frontend Data Flow:
  1. User action in component
  2. React Query mutation triggered
  3. API client (Axios) adds auth token
  4. Sent to FastAPI endpoint
  5. Backend validates with Pydantic
  6. Service executes business logic
  7. Database updated
  8. Response returned
  9. React Query cache invalidated
  10. Component re-renders with new data

State Management:
  • React Query: Server state (data, caching, sync)
  • AuthContext: Auth state (token, userId)
  • Component state: Local UI (forms, modals)
